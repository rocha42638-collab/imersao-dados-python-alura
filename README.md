Durante a imersão, foi realizada uma análise de dados utilizando uma
base com mais de 130 mil linhas (133349 rows × 11 columns), passando por
todas as etapas principais de um projeto de análise de dados.

Aprendizados principais:

Durante esta imersão foi possível entender, na prática, o fluxo de um
projeto de dados: 1. Importar os dados 2. Limpar e tratar os dados 3.
Analisar os dados 4. Visualizar os dados 5. Gerar insights

Insights Obtidos

- Profissionais seniores possuem maior média salarial anual.
- A maioria das vagas está concentrada em níveis Pleno e Sênior.
- O trabalho remoto apresenta variação salarial dependendo da senioridade.
- Cargos executivos possuem a maior média salarial do dataset.

Este projeto representa não apenas o aprendizado de comandos em Python,
mas principalmente o desenvolvimento do raciocínio analítico para
trabalhar com dados reais.

Conclusões

A análise dos dados permitiu observar padrões importantes no mercado de trabalho na área de dados. 
Profissionais com maior nível de senioridade apresentam maiores médias salariais, enquanto cargos executivos concentram os maiores salários. 
Também foi possível observar diferenças salariais entre países e regimes de trabalho remoto, mostrando como a localização e o modelo de trabalho influenciam a remuneração na área de dados.
